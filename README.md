# Simon-Says-Game

In this game, Simon will randomly pick a color and all you have to do is follow what Simon does. Simon's goal is to try to eliminate. See if you can keep up with what Simon does! 

<b>Technologies</b>: 
- HTML
- CSS
- JavaScript 
- jQuery
